# Challenge Intermedio: Modelado con Spark ML

## Descripción

Este proyecto forma parte de un challenge intermedio sobre **PySpark** y **Spark ML**, cuyo objetivo fue descargar, preparar y modelar la base de datos **Wine Quality Dataset** desde el UCI Machine Learning Repository.

El enfoque principal del reto fue aplicar técnicas de *machine learning* usando **Spark ML**, trabajando con datos a escala y generando un modelo predictivo básico.

## Objetivos del proyecto

* Descargar y cargar los datos del Wine Quality Dataset en PySpark.
* Realizar limpieza y preprocesamiento de las variables.
* Crear un vector de características con Spark ML.
* Entrenar un modelo de **Regresión Logística** para predecir la calidad del vino.
* Generar métricas y gráficas de rendimiento del modelo.

## Tecnologías y librerías utilizadas

* Python
* **PySpark / Spark ML**
* Pandas
* Matplotlib

## Contenido del repositorio

* `notebook_modelo_spark.ipynb`: notebook con el análisis, el preprocesamiento, la construcción del modelo y las visualizaciones.
* `README.md`: documentación del proyecto.

## Notas

Este proyecto forma parte de un módulo formativo sobre análisis y modelado de datos en PySpark, enfocado en llevar al estudiante de operaciones básicas de datos hacia el uso de Spark ML para clasificación.
